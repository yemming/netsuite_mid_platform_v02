## 📋 測試前準備

### 1. 環境檢查
- [ ] 確認 Node.js 已安裝（建議 v18+）
- [ ] 確認專案依賴已安裝：`npm install`
- [ ] 確認 Tailwind CSS 配置已更新（包含 SCM 資料夾路徑）

### 2. 啟動開發伺服器
```bash
npm run dev
```

### 3. 訪問系統
- 開啟瀏覽器訪問：`http://localhost:3000/scm` 或根據你的路由設定
- 確認首頁正常顯示

   - [ ] 看到下拉選單「選擇事件」
   - [ ] 看到「取消」和「確認」按鈕

#### 1.6.3 選擇事件
1. 點擊下拉選單
2. **預期結果：**
   - [ ] 看到事件列表（如：聖誕節促銷、春節假期、雙十一促銷）

   - [ ] 選中事件顯示在下拉選單中
   - [ ] 「確認」按鈕變為可點擊狀態

#### 1.6.4 確認關聯

2. **預期結果：**
   - [ ] 彈窗關閉

#### 1.6.5 取消操作
1. 重新開啟彈窗
2. 點擊「取消」按鈕
3. **預期結果：**
   - [ ] 彈窗關閉
   - [ ] 沒有執行任何關聯操作

#### 1.6.6 未選擇事件時確認

3. **預期結果：**
   - [ ] 「確認」按鈕為禁用狀態（灰色，不可點擊）
